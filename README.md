## 🚀 ¿Qué es Aetherium?
Aetherium no es solo un software de contabilidad; es un **Sistema Operativo Neuronal** que centraliza todas las operaciones críticas de una empresa moderna bajo una arquitectura de nube híbrida.

### 🧠 Capacidades Core:
*   **Neural Call AI**: Interfaz de voz bilingüe (Español/Inglés) con procesamiento de lenguaje natural para ejecutar comandos operativos mediante voz.
*   **Motor Fiscal Soberano**: Integración profunda con el ecosistema de la **DGII** (República Dominicana), incluyendo validación de RNC, gestión de NCF y generación automática de formatos 606/607.
*   **Lex-DB (Biblioteca Legal)**: Base de conocimientos inteligente que procesa leyes, contratos y normativas en tiempo real mediante IA.
*   **Marketplace Red**: Ecosistema de comercio integrado para activos de alto valor y gestión de inventario inteligente.
*   **Auditoría Cruzada**: Sistema preventivo que utiliza algoritmos neuronales para detectar inconsistencias fiscales antes de que se conviertan en multas.

## 🛠️ Stack Tecnológico
Para garantizar una disponibilidad del 100% y una seguridad de grado militar, Aetherium utiliza:
*   **Frontend**: React 19 + Tailwind CSS (Desplegado en Netlify).
*   **Cerebro (Backend)**: Node.js / Express (Desplegado en Render).
*   **Base de Datos**: PostgreSQL vía Supabase con seguridad **RLS (Row Level Security)**.
*   **IA**: Groq Neural Llama 3 & Whisper para procesamiento de voz instantáneo.
*   **ORM**: Prisma para integridad de datos absoluta.

## 🛡️ Seguridad & Privacidad
El sistema implementa cifrado de extremo a extremo y políticas de aislamiento de datos que garantizan que la información fiscal y comercial sea accesible únicamente por el propietario soberano de la entidad.
